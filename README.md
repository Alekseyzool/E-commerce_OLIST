# E-commerce OLIST

## Описание проекта
Аналитический разбор маркетплейса Olist: от загрузки исторических CSV в базу данных до когортных исследований и приоритизации продуктовых гипотез на основе юнит-экономики.

## Цель
Понять причины стагнации выручки и предложить продуктовые эксперименты, которые снизят отмены и вернут рост повторных покупок.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `SQLAlchemy`, `tqdm`, `python-dotenv`

## Функциональность
- Скрипт импорта CSV в PostgreSQL/SQLite с проверкой схемы и документацией таблиц.
- Когортный анализ удержания, расчёт LTV/GMV и сегментация пользователей.
- ICE-ранжирование гипотез с подбором guardrail-метрик для A/B-тестов.
- Детальные визуализации и выводы для продуктовой команды.

## Инструкция по запуску
1. Скопируйте `.env` по инструкции из ноутбука и задайте параметры подключения к базе (PostgreSQL/SQLite).
2. Установите зависимости: `pip install -r requirements.txt`.
3. Запустите `E-commerce OLIST DB.ipynb`, чтобы подготовить инфраструктуру и проверить целостность данных.
4. Откройте `E-commerce OLIST analysis.ipynb` и выполните ячейки последовательно для воспроизведения аналитики.
